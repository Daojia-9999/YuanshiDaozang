# 續黃白鏡

京口夢覺道人李文燭晦卿甫著

## 醒醒歌
石函記，漁荘錄，兩公留下金丹目，胡為今古錬丹家，個個無成空碌碌。

玄律嚴，天威肅，金丹原是神仙錄，肯教容易與凡夫，妄干徒受天之戮。

求此事，須積德，務要把心先放直，陰德積多天自予，天予不教人費力。

與君說，休疑惑，莫向凡銀討花色，硃砂縱死亦何用，枉把黑鉛錬枯極。

休輕信，方上客，山中尋草煮八石，更將水銀投黑鉛，草煅硃砂真可惜。

要汞乾，須火息，先要將紅飛入黑，紅為火神黑水氣，神氣相交自相得。

神居南，氣居北，南北之中討消息，結成龍虎兩胞胎，借爾胞胎將永翼。

龍虎胎，真命脈，誰人認得誰會摘，有人問我胞胎形，射日紅塵滾窗隙。

浣此塵，出水國，全憑烈火將他逼，逼他出水號天毓，天毓乾汞神仙則。

神既凝，天毓熟，不怕水銀心不伏，水銀見毓登時乾，從此河車任反覆。

火是神，汞是谷，汞火一家親骨肉，火若不凝汞不乾，汞乾方可名基築。

水銀乾，色潔白，還要將他來錬赤，汞銀錬赤造黃輿，直上蓬萊無阻隔。

死硃砂，枯骨革，凡夫寶之如拱壁，大家稱此作天毓，惹得神仙暗點額。

暗點額，笑人差，天毓原是砂精華，因到鉛關行一轉，玉陽呼此作黃芽。

砂之精，鉛之華，結就天毓在坎家，不向此中求秘訣，鐵鞋踏破遍天涯。

遍天涯，沒處尋，坎中一點天地心，此心原是神火化，惟此能除汞裏陰。

汞裡陰，名為癸，癸若乾時汞自死，若要癸乾也無難，坎中一點潛龍骸。

潛龍髓，即天毓，不比尋常渣質流，砂鉛未有先有此，莫把砂皮石壳求。

死硃砂，石骷髏，骷髏裡面神氣休，可憐神氣不留住，反把骷髏當寶收。

笑世人，忒煞癡，只在丹房弄鼎池，硃砂黑鉛真鼎器，陰陽交合即成兒。

造天毓，生身同，鉛受砂施片晌工，雜類一毫原不用，自然一點篏當中。

說交媾，自有期，莫教太早並太遲，若不得時空合體，一陽纔起正當時。

天毓花，有兩朵，一罏實無兩樣火，譬如懷胎有女男，莫怪吾言太瑣瑣。

太瑣瑣，忒明明，天毓不是無因生，只為有形相合體，一副黃芽兩地萌。

砂中汞，最陰柔，活則沉兮死則浮，萬法千方皆不死，一見天毓癸便抽。

汞見毓，陰氣滅，清如霜色鬆如雪，不來而舍求乳母，依舊前途還未徹。

汞實死，即金丹，須知栽接有何難，起根轉制清真極，久久人餐生羽翰。

## 水心篇
憶昔逢師在酒樓，一時魚水便相投，他言下手無根蒂，且籍凡鉛起焅頭。

卷卷丹書說死砂，硃砂縱死亦還差，能將硃裡砂來死，方是丹房老作家。

硃裡求砂始是砂，此砂凝結即黃芽，人人都把硃砂死，誰肯回思自己差。

硃裡水銀名己土，砂為神火土之娘，古今多少老罏火，不識娘兒空自忙。

要死兒時先死娘，不將娘死子難亡，娘逢壬水方纔死，誰解鉛中是法王。

養砂之法最辛勤，鼎要溫溫火要文，此處莫教差錯了，自然水火氣氤氳。

砂到鉛中造化生，丁壬相合結精英，開罏慢自分龍虎，入鼎先須定甲庚。

庚為白虎甲為龍，虎在西方龍在東，兩獸捉來歸己汞，一輪明月正當中。

甲龍庚虎兩胞胎，借爾胞胎把汞培，但得虎龍同入汞，何愁癸水不分開。

癸水分明汞裡生，一逢戊土便相迎，如今睡醒纔方覺，戊土原來即甲庚。

須知乾汞也無難，戊癸相逢汞自乾，戊癸化成瑩惑去，汞銀一味即金丹。

汞若乾時即白金，白金猶自怕含陰，鋉成紫赤真金體，留在丹房捕赤禽。

水銀乾徹即枯鉛，世上枯鉛豈足言，汞沾半點枯鉛氣，便覺癡呆軟似綿。

癸鉛枯鉛露先天，枯盡鉛時汞體堅，記得癸鉛初見戊，嗷嗷泣泣似鳴蟬。

庚金胎處癸長生，戊土來時癸自斃，癸盡自然金現象，何勞人去費神情。

書傳乳哺在西隣，天下丹房說是銀，坤母與西纔肘壁，凡銀卻與汞何親。

乳娘沒乳且休言，枉把真鉛裡面煎，此處不知栽接法，依然前路又茫然。

拙拙時嘗到我前，從來口吃不多言，今朝問我真鉛理，汞不乾時不是鉛。

又問天毓事若何，人間渣質總成訛，天毓不是尋常物，生長西南造化窩。

先種天毓在黑鉛，從來此處不輕傳，誰知先把天毓種，會種天毓便是仙。

天毓正是方生物，元始初成造化根，新月一鈎相仿佛，俗人半句莫評論。

硃砂因與黑鉛交，一點神光射入胞，從此天毓纔有象，昔人稱此作初爻。

硃砂交後體無妨，只用中間一線光，譬如男女相交後，肢體何曾略損傷。

千方萬法把砂為，神氣精亡剩死尸，多少道人皆着假，欲將枯骨養孩兒。

一罏火隱一罏丹，火隱何愁汞不乾，成始成終惟火汞，一毫凡雜沒相干。

天毓初用黑鉛栽，終是人間濁胚胎，若把汞金為鼎器，自然生長是靈材。

天毓出產汞金中，力量原來自不同，八石人教隨意養，也無一樣不成功。

笑殺人間井底蛙，盤山度嶺覓仙花，將來搗汁同砂煮，汞走毓飛路愈賖。

總識天毓是汞鉛，不知超脫也徒然，大都超脫皆憑火，只在天毓聚散邊。

結得毓胎在坎中，一時辭母入東宮，神仙說道無他法，些子玄微在火功。

汞藉天毓癸氣收，天毓與癸共綢繆，全憑火力攻他出，野馬氤氳片片浮。

丁壬妙合胎須脫，戊癸天然火要超，火散汞乾超出世，脫然無累自逍遙。

神凝點汞汞成銀，汞點成銀復養神，神養聖靈仍點汞，一番超過一番新。

火是神兮汞是精，精為乙木火為丁，若非己汞親枝葉，誰敢無知汞裡行。

靈臺不敢說清澄，心上猶貪最上乘，鉛鼎有壬今既識，硃池丁火豈難凝。

硃硫自識為神火，己汞誰知是冷砂，己汞始乾稱白雪，硃硫初結即黃芽。

錬丹起手要清真，根不清真枉費神，多少盲燒瞎煉士，嘵嘵只是講凡銀。

東海火庵藏聖汞，西山泉眼匿真鉛，須向鼎中細分剖，免使罏邊又禍僭。

丁寍錬士要防危，最怕人妖暗裡窺，丹室四維懸寶鏡，洪罏當頂覆儲帷。

只恐凡夫意不識，人心一正即誠明，任教無限妖魔鬼，盡化丹房護法兵。

丹房伴侶要同心，伴侶同心可斷金，若是同心人做事，阿誰魔鬼敢相親。

壇場不可近墳瑩，古廟神堂雞犬聲，龍抱虎超坐生氣，之玄兩字水相迎。

丹房門戶莫當中，坎宅開門在巽風，離震地天山換澤，能依此法自亨通。

竈支鼎立張廉處，鬼盜神偷不可當，伴侶弗知時吵閙，火災官事禍非常。

胡宗明是最凶徒，此處安罏禍不無，寄語丹房掌罏者，莫將微細反心精。

王文王用兩星辰，仔細評來總不仁，不若貪狼並武巨，三星之上可相親。

又把三星校短長，雷風水火用貪狼，乾坤山澤偏宜武，便用天醫也不妨。

識得臨罏秘密機，不愁雞犬不隨飛，如今且到龍光寺，看過龍沙即便歸。

禮罷真君就轉槎，閒雲留我應龍沙，不知南國煙霞士，自有龍沙種在家。

自在滕王閣下歸，如今還只掩柴扉，三餐子罷無些事，鎮日窗前寫化機。

## 黃白鏡識
昔余遭劉青田累，幾成孔北海禍，賓客皆散去，惟姑蘇拙老獨不去。已而事白，執侍如故，由是余多老，遂欲以修錬胎仙之法告之，第憚其性根欠利，卒不能悟，故續此鏡，先以嘗其聰明，俟其觸類旁通，能悟本心，然後有以教之，不然徒泄天寶，何益也。六祖大師云：不識本心，學法無益。老其勉之哉，老其勉之哉。

萬歷辛丑午月京口夢覺道人李文燭晦卿甫識。
